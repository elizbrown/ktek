KTEK
[![Build Status](https://travis-ci.org/tmlbl/ktek.svg?branch=master)](https://travis-ci.org/tmlbl/ktek)
====

KTEK is a framework for generating technical books for the Amazon Kindle. The
goal is to allow the author to write GitHub-flavored markdown, one file per
chapter, and have all of the nasty XML and such generated for Kindle.

This project is currently in early-stage development.
